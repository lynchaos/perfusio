Excited to share **perfusio** — an open-source Python library that
reproduces the self-driving perfusion methodology of Gadiyar et al. (2026,
*Biotechnology & Bioengineering*).

🔬 **What it does:**
• Stepwise GP hybrid model (SW-GP) for CHO cell culture
• Entity-embedding transfer learning across cell lines (Hutter et al. 2021)
• 11 Bayesian acquisition functions (EI, LogEI, qNEHVI, …)
• Online digital twin with OPC UA + SQL connectors
• Reproducible paper figures (Fig. 4, 6, 7, 8)

📦 **Get it:**
```
pip install perfusio
python examples/reproduce_paper_figures.py
```

🔗 https://github.com/your-org/perfusio
